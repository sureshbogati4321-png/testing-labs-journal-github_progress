
## Limits of Binary Representation
**Determine the limits of binary representation:**
   - Largest positive number with 8 bits in unsigned binary
   - Largest positive number with 8 bits in signed binary (two's complement)
   - Smallest negative number with 8 bits in signed binary (two's complement)
1) Largest positive number (8-bit unsigned)

Unsigned 8-bit ranges from 0 to 255

 11111111₂ = 255₁₀

2) Largest positive number (8-bit signed, two’s complement)

Signed 8-bit range is −128 to +127

 Largest positive value:
01111111₂ = 127₁₀

3) Smallest negative number (8-bit signed, two’s complement)

 Smallest (most negative) value:
10000000₂ = −128₁₀

Summary Table
Case	Binary	Decimal
Largest unsigned (8-bit)	11111111	255
Largest signed (8-bit)	01111111	127
Smallest signed (8-bit)	10000000	-128
