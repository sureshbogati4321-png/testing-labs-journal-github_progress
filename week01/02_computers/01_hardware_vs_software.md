# Part 1 – Hardware vs Software
Hardware is the physical parts of a computer that you can touch, such as the keyboard, monitor, CPU, and hard drive.
Software is the programs and instructions that tell the hardware what to do, such as operating systems, games, and web browsers.
## Explain Hardware
Write **at least 50 words** explaining what hardware is.
Hardware refers to the physical components of a computer or electronic device that can be seen and touched. It includes devices such as the CPU, keyboard, mouse, monitor, printer, and storage drives. Hardware works with software to process data, perform tasks, and enable user interaction.

## Explain Software
Write **at least 50 words** explaining what software is.
Software is a collection of programs, instructions, and data that enables a computer or device to perform specific tasks. It includes system software, such as operating systems, and application software, such as word processors and games. Software directs hardware operations and supports user activities across diverse digital environments and industries.
