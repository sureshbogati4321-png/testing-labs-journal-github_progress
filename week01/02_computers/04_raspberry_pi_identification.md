| Component | Function |
|-----------|----------|
| SoC |The main integrated circuit that contains the CPU, GPU, memory controller, and other essential processing components. It acts as the "brain" of the device. |
| GPIO |A set of programmable pins used to connect and control external electronic components such as LEDs, sensors, motors, and switches. |
| USB |Ports used to connect peripheral devices like keyboards, mice, flash drives, cameras, and other USB accessories. |
| HDMI |Provides digital video and audio output to a monitor, television, or projector. |
| Power |The connector through which the board receives electrical power required for operation. This may be a USB-C or micro-USB port, depending on the model. |
| microSD |A slot for a microSD card, which is commonly used to store the operating system, applications, and user files. |
| Network |Usually refers to the Ethernet port, enabling a wired connection to a local network or the internet. |
| Wireless |Built-in wireless technologies such as Wi-Fi and Bluetooth that allow cable-free internet access and communication with compatible devices. |

## Reflection
Why are many components integrated into a single board in embedded computers such as the Raspberry Pi?
Write **100 words**.
Many components are integrated into a single board in embedded computers such as the Raspberry Pi to reduce size, cost, and power consumption. Combining essential parts like the processor, memory interfaces, communication ports, and wireless modules makes the system compact and efficient. This design simplifies assembly and improves reliability because fewer external connections are required. It also makes the device easier to use for education, prototyping, and commercial applications. By providing most functions on one board, embedded computers can perform a wide range of tasks while remaining affordable, portable, and suitable for dedicated purposes.
